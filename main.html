FILLER

Filler is used to fill your information in input fields.

How to use :
 
 # Save data :
   
   (1. Press Edit button 
   (2. Fill a unique token number(All information is linked to this token number)
   (3. Fill all information
   (4. Press save button 
 
 # Fill data :
   
   (1. Fill token number(You can use information linked to this token number) 
   (2. Fill pattern sequence wise(Let you have input in sequence username, email, number 
       so you write uex in pattern) 
   (3. Patterns 
       username -> u
       firstname -> f
       lastname -> l
       name -> n
       email -> e 
       father's name -> y
       mother's name -> z
       dob -> j
       Mobile number -> x
       college -> c
       address -> a
       adhar Number -> d
       blank -> b
   (4. Press set

 # For practice and watch your data 
   
   (1. https://goo.gl/7XMUTq
   (2. Copy pattern and paste it
   (3. press set
